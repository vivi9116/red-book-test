# 你喜欢一个人，是想被偏爱还是被理解？

封面主标题:
你喜欢一个人，是想被偏爱还是被理解？

封面副标题/角标:
10题测一测

账号角标:
懂你的X学姐

小红书发布标题:
10题测测你喜欢一个人时最需要什么

正文:
有些人一心动，就想确认自己是不是被坚定选择。
也有些人最在意的不是热烈，而是“你懂我”。

这套是免费小测试，不用想太复杂。
每题选最像你的一项：

A = 3分
B = 2分
C = 1分

做完10题后，把分数加起来。

10-15分：更想被理解

16-23分：两种都需要

24-30分：更想被偏爱

评论区留下：
「我XX分，最像第X、X、X题」

我按你的分数和题号，回复你更需要哪一种喜欢。

置顶评论:
别只报分数，也说一下你最像哪几题，我会按分数和题号一起看。

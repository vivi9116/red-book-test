# 边界感模式测试｜小红书长文发布文案

小红书发布标题：
你不是不会拒绝，你是太怕一拒绝就失去关系

正文：
有些人不是没有边界。

只是每次边界快要出现的时候，心里会先冒出很多声音：

“TA会不会觉得我很冷漠？”

“是不是就这一次，忍忍算了？”

“我不答应，会不会显得我很自私？”

“可是我真的已经很累了。”

于是你又接住了别人的请求、别人的情绪、别人的临时安排。

表面上关系没有冲突，事情也被你处理好了。

但你心里慢慢开始有一种说不出的委屈：

为什么每次都是我理解别人？

为什么我一拒绝，就像自己做错了？

为什么我已经很累了，还要先照顾别人的感受？

边界感不是让你变冷，也不是让你突然强硬。

它更像一条关系里的红线：

我可以在乎你，但我也需要保留自己。

我可以理解你，但这不代表我要全部承担。

我可以温柔，但温柔不等于没有限制。

这个测试会从 6 个维度看你的边界模式：

请求压力、愧疚反应、冲突承受、自我空间、责任分清、表达修复。

你会看到自己更像：

过度承担型、愧疚软化型、冲突回避型、高墙防御型、自我消音型，还是清晰守界型。

不是给你贴标签，而是帮你看清：

你在关系里，是从哪里开始把自己交出去的。

CTA：
不是让你变得不好相处，而是帮你练习温和、清楚、稳定地把自己放回关系里。

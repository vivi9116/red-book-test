# 边界感模式测试｜小红书长文文案

商品标题：
边界感测试｜你是过度承担，还是一忍就筑墙？

小红书发布标题：
你不是不会拒绝，你是太怕一拒绝就失去关系

正文：
你不是不会拒绝。

你只是太怕一拒绝，关系就变坏。

很多时候，你明明已经不舒服了，却还是会先想：

“我这样说，会不会太冷漠？”

“TA现在需要我，我是不是不该拒绝？”

“算了，就这一次吧。”

于是你一次次接住别人的请求、情绪和临时安排。

表面上关系还算平静，心里却越来越累。

边界感不是让你变得不好相处，也不是让你突然强硬。

它是在关系里看清楚：

哪些是我愿意承担的。

哪些是我可以拒绝的。

哪些情绪不需要全部由我负责。

哪些空间，我也需要留给自己。

这个测试会从 6 个核心维度分析你的边界模式：

请求压力、愧疚反应、冲突承受、自我空间、责任分清、表达修复。

你会看到自己更容易在哪一条线松动：

是过度承担，还是愧疚退让。

是害怕冲突，还是习惯把自己放到最后。

测完后会生成你的专属边界报告，帮你看清主类型、副倾向、触发场景和第一步练习。

CTA：
不是让你变得不好相处，也不是给你贴标签，是帮你看清你在关系里重复的模式，把自己重新放回关系里。

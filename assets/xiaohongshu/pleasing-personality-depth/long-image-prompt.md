# Long-Form 引流配图 Prompt

## Purpose

这张图服务于小红书长文引流，不是测试封面。它要让用户停下来读正文，情绪更安静、更像一页手帐文章开头。

不要给出完整测试答案，不要展示 6 种完整分型，不要直接列出报告内容，不要让用户看完图片就觉得已经知道结果。

## Style

长文引流配图，延续今天的小红书手帐拼贴风：暖白纸质背景、粉红和莓红点缀、轻胶带、手写箭头、便签、柔和关系线、小爱心、小星星。整体比封面更克制、更安静，像一页被认真整理过的情绪笔记。

不要真人写实孤独照片，不要暗色卧室，不要医学诊断感，不要恐吓式视觉。

## Layout

4:5 vertical image.

Top small label:

```text
写给总是先照顾别人的你
```

Main sentence, large but softer than cover:

```text
你不是不会拒绝，
你是太害怕关系变冷。
```

Middle visual:

- A warm paper note with a small relationship map
- One tiny message bubble: "没事，我可以"
- One crossed-out tiny line: "我其实不想"
- A soft arrow pointing to a question mark sticker

Bottom small sentence:

```text
真正重要的是：那我是哪一种讨好？
```

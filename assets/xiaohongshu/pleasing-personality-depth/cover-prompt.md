# Cover Prompt

## Style

小红书原生测试封面风，手帐拼贴感，暖白纸质背景，轻微纸张纹理，粉红与莓红标题点缀，少量胶带、爱心、星星、手绘箭头、贴纸边框。画面可爱但不幼稚，像女生会收藏的心理测试封面。

不要真人写实孤独照片，不要电影感摄影，不要阴暗蓝色卧室，不要医疗诊断感，不要复杂解释文字。

## Layout

4:5 vertical cover.

Top small label:

```text
超准关系模式测试
```

Main title, large and highly readable:

```text
你是哪一种讨好型人格？
```

Center visual:

- Four soft paper cards or sticker blocks
- Each card has one short option-style phrase:
  - A 怕别人不高兴
  - B 不敢拒绝请求
  - C 总先反省自己
  - D 被需要才安心
- Add small hand-drawn lines connecting the cards to a central heart/map motif

Bottom note:

```text
测完看见你在关系里反复牺牲的模式
```

## Image Prompt

Create a Xiaohongshu native psychological quiz cover, vertical 4:5, warm white paper texture background, hand-account collage design, soft pink and berry red accents, cute but refined stickers, washi tape, small hearts, stars, hand-drawn arrows, paper cards, central relationship map heart motif. Large Chinese title: "你是哪一种讨好型人格？". Small top label: "超准关系模式测试". Four option cards: "A 怕别人不高兴", "B 不敢拒绝请求", "C 总先反省自己", "D 被需要才安心". Bottom note: "测完看见你在关系里反复牺牲的模式". Clean readable typography, high contrast, polished Xiaohongshu cover style, no photorealistic person, no dark bedroom, no medical diagnosis feeling.

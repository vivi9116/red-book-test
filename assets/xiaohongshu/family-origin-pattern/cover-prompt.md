# Cover Prompt

## Style

小红书原生测试封面风，手帐拼贴感，暖白旧纸纹理，低饱和灰绿、旧照片棕、柔和砖红点缀。画面像一页“成长关系档案”：旧照片边框、手写便签、家庭关系线条图、小胶带、手绘箭头、轻微复古印章。可爱但不幼稚，温柔、有收藏感。

不要真人写实孤独照片，不要阴暗卧室，不要创伤恐吓，不要医疗诊断风。

## Layout

4:5 vertical cover.

Top small label:

```text
超准原生家庭测试
```

Main title:

```text
你的原生家庭
正在影响哪种关系模式？
```

Center visual:

- 一个小小的“小时候的我”剪影或贴纸坐在中心
- 周围 6 张便签纸，像关系地图一样连线
- 便签文字：
  - A 讨好求稳
  - B 情绪警觉
  - C 缺爱确认
  - D 回避封闭
  - E 自责修复
  - F 控制安全

Bottom note:

```text
测完看见你在关系里重复的旧脚本
```

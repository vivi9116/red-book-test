# 小红书引流长文

你有没有发现，有些关系里的反应，明明你不想这样，却总是自动发生。

对方一沉默，你就开始紧张。

别人不高兴，你会先检查自己。

你很难直接说需要，也很难相信爱是稳定的。

有时候你也知道，现在这段关系并没有真的发生什么大事。

但身体好像比理智更快一步。

它已经开始复盘、预判、退让、解释，或者把自己关起来。

这不一定是你太敏感。

也不一定是你不成熟。

很多时候，这是一个人很早学会的关系脚本。

小时候，如果你发现“懂事一点，关系就稳定一点”，长大后你可能会习惯讨好求稳。

如果你发现大人的情绪总是突然变化，你可能会变得很会察言观色。

如果你很少确定自己会被稳定地选择，你可能会在亲密关系里反复确认。

如果需求经常落空，你可能会学会不说、不期待、先退回自己那里。

原生家庭影响你的，不一定是某一件很大的事。

更可能是它让你形成了一套自动反应：

我该怎么靠近别人？

我能不能表达需要？

关系变冷时，是不是我的错？

我是不是只有足够懂事，才值得被爱？

所以真正重要的问题不是“我的原生家庭有没有问题”。

而是：

那我是哪一种关系脚本？

我做了一个「原生家庭关系模式测试」。

它不是为了责怪父母，也不是为了给你贴标签。

它只是帮你看清：

你在关系里最常重复哪一种旧模式；

它可能是怎么形成的；

它会在什么场景里被触发；

以及你可以从哪一步开始，不再完全被过去的反应牵着走。

不是为了证明你受过伤。

是为了让你终于知道，为什么自己总是在某些关系里变回小时候的样子。

如果你也想知道：

“我的原生家庭，到底塑造了我哪一种关系模式？”

可以从这次测试开始。

# Long-Form 引流配图 Prompt

## Purpose

这张图服务于小红书长文引流，不是测试封面。它要让用户在读正文前先进入一种“这可能说的是我家”的状态。

不要给出完整测试答案，不要列出全部关系脚本，不要把报告模块直接展示完。免费图只负责命名疼痛和打开问题，付费测试负责给出具体分型、来源解释和改变路径。

## Style

长文引流配图，延续今天选定的小红书手账拼贴风，但要比封面更安静、更像一页被认真整理过的成长笔记。

画面使用旧相册纸、浅米白底、低饱和灰绿、淡粉便签、半透明胶带、手写批注、细线箭头、家庭关系小地图、轻微纸张折痕。整体温柔、克制、有记忆感，不做阴暗创伤视觉。

不要真人写实孤独照片，不要恐吓式家庭裂痕，不要医学诊断感，不要把父母塑造成绝对反派。

## Layout

4:5 vertical image.

Top small label:

```text
写给总是在家里先懂事的你
```

Main sentence, large but calmer than cover:

```text
你不是突然变敏感，
你只是太早学会了看脸色。
```

Middle visual:

- A soft paper family map with three small circles connected by thin lines
- One tiny sticky note: "我没事"
- One folded old-photo corner with the words: "小时候的规则"
- A dotted arrow pointing toward a question mark sticker

Bottom small sentence:

```text
真正重要的是：那我重复的是哪一种关系脚本？
```
